Password Reset
Password reset is only available while this browser is connected through the device's default soft AP mode. You may reset the password, doing so will:
  • unpair and erase all Bluetooth devices
  • erase all Wi-Fi configurations
  • erase all cloud destinations
  • change the encryption key for future file encryption
  • require you to change the password on any servers or tools used to decrypt the files
Please use at least 6 characters, include at least one letter and at least one number, and do not use spaces.
